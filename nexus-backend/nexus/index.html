Setup & Authentication
Everything you need to get NEXUS running with real Google APIs
Setup steps
Install: pip install -r requirements.txt
Google Cloud Console → enable Gmail API + Calendar API → OAuth2 Desktop → download credentials.json
Copy .env.example → .env · add GROQ_API_KEY (free at console.groq.com)
uvicorn main:app --reload --port 8000
Click "Connect Google" → one-time OAuth consent → token.json saved
NEXUS is live — polls Gmail every 30s, LangGraph handles everything
